# Projet compétences
## Présentation
Ce projet est un projet de fin de formation de la Wild Code School de Nantes. Il a pour but de mettre en pratique les compétences acquises par les membres d'équipes.

## Prérequis
- Java 11+
- docker
- docker-compose
- un IDE (IntelliJ, Eclipse)
